# nacos-study
nacos study
### 研究一下微服务注册中心开源方案，搭建一个springcloud开发环境，完成一个helloworld微服务开发，并把该服务注册到nacos上
前置条件：启动nacos服务，配置暂时都用默认的
访问地址 http://127.0.0.1:8080/helloworld
